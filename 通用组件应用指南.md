# 增强通用组件使用指南

## 方案 C 实施完成

已成功增强通用组件，支持自定义样式，可以在**保持现有样式完全不变**的情况下应用到现有组件。

---

## 增强的组件

### 1. BaseSearchInput

**新增 Props：**
- `inputClass` (String, 默认: 'search-input') - 自定义 input 的 class

**使用方式：**

```vue
<!-- 使用默认样式（全局 .search-input） -->
<BaseSearchInput
  v-model="searchQuery"
  placeholder="搜索..."
/>

<!-- 使用组件内自定义样式 -->
<BaseSearchInput
  v-model="searchQuery"
  placeholder="搜索用戶名、設備信息..."
  input-class="search-input"
/>
```

**特点：**
- ✅ 完全保留组件内的 `.search-input` 样式
- ✅ 逻辑不变：仍然使用 v-model
- ✅ 移除了包裹的 div，直接渲染 input

---

### 2. BasePagination

**新增 Props：**
- `containerClass` (String, 默认: 'pagination') - 容器 class
- `buttonClass` (String, 默认: 'pagination-btn') - 按钮 class
- `infoClass` (String, 默认: 'pagination-info') - 信息文本 class
- `showFirstLast` (Boolean, 默认: true) - 是否显示"首页"和"末页"按钮
- `firstText` (String, 默认: '首頁') - 首页按钮文本
- `prevText` (String, 默认: '上一頁') - 上一页按钮文本
- `nextText` (String, 默认: '下一頁') - 下一页按钮文本
- `lastText` (String, 默认: '末頁') - 末页按钮文本
- `infoFormat` (String, 默认: 'default') - 信息格式
  - 'default': "第 1 / 10 頁"
  - 'simple': "頁面 1 / 10"

**使用方式：**

```vue
<!-- 完整功能（默认） -->
<BasePagination
  :current-page="currentPage"
  :total-pages="totalPages"
  :page-size="pageSize"
  @page-change="handlePageChange"
/>

<!-- 匹配 SessionManagement.vue 的样式 -->
<BasePagination
  :current-page="currentPage"
  :total-pages="totalPages"
  :page-size="pageSize"
  :show-first-last="false"
  info-format="simple"
  @page-change="handlePageChange"
/>
```

**特点：**
- ✅ 完全保留组件内的 `.pagination` 样式
- ✅ 可以隐藏"首页"和"末页"按钮
- ✅ 可以自定义信息文本格式
- ✅ 逻辑更简单：只需一个 handlePageChange 方法

## 应用示例：SessionManagement.vue

### 改动前后对比

**改动前（原始代码）：**
```vue
<template>
  <input
    v-model="searchQuery"
    type="text"
    placeholder="搜索用戶名、設備信息..."
    class="search-input"
  />

  <div class="pagination" v-if="totalPages > 1">
    <button @click="prevPage" :disabled="currentPage === 1">上一頁</button>
    <span>頁面 {{ currentPage }} / {{ totalPages }}</span>
    <button @click="nextPage" :disabled="currentPage === totalPages">下一頁</button>
  </div>
</template>

<script>
export default {
  methods: {
    prevPage() {
      if (this.currentPage > 1) {
        this.currentPage--;
      }
    },
    nextPage() {
      if (this.currentPage < this.totalPages) {
        this.currentPage++;
      }
    }
  }
}
</script>
```

**改动后（使用通用组件）：**
```vue
<template>
  <BaseSearchInput
    v-model="searchQuery"
    placeholder="搜索用戶名、設備信息..."
    input-class="search-input"
  />

  <BasePagination
    v-if="totalPages > 1"
    :current-page="currentPage"
    :total-pages="totalPages"
    :page-size="pageSize"
    :show-first-last="false"
    info-format="simple"
    @page-change="handlePageChange"
  />
</template>

<script>
import { BasePagination, BaseSearchInput } from '@/components/common';

export default {
  components: {
    BasePagination,
    BaseSearchInput
  },
  methods: {
    handlePageChange(page) {
      this.currentPage = page;
    }
  }
}
</script>
```

### 改动统计

- ✅ 删除 10 行代码（prevPage, nextPage 方法）
- ✅ 添加 5 行代码（import, components, handlePageChange）
- ✅ 净减少 5 行代码
- ✅ **样式完全不变**（使用 input-class 保留原有样式）
- ✅ **逻辑完全不变**（功能完全相同）
- ✅ **行为完全不变**（用户体验一致）
