## 改动 2: 替换分页（第 70-74 行）

### 改动前：
```vue
<div class="pagination" v-if="totalPages > 1">
  <button @click="prevPage" :disabled="currentPage === 1">上一頁</button>
  <span>頁面 {{ currentPage }} / {{ totalPages }}</span>
  <button @click="nextPage" :disabled="currentPage === totalPages">下一頁</button>
</div>
```

### 改动后：
```vue
<BasePagination
  v-if="totalPages > 1"
  :current-page="currentPage"
  :total-pages="totalPages"
  :page-size="pageSize"
  @page-change="handlePageChange"
/>
```

**说明：**
- 逻辑不变：仍然使用 `currentPage`, `totalPages`, `pageSize`
- 需要添加一个方法：`handlePageChange(page) { this.currentPage = page; }`
- 样式自动应用：BasePagination 已包含 `.pagination` 样式
- 功能增强：自动添加"首页"和"末页"按钮

## 改动 5: 添加 handlePageChange 方法

### 在 methods 中添加：
```javascript
methods: {
  handlePageChange(page) {
    this.currentPage = page;
  },
  // ... 其他现有方法保持不变
}
```

**说明：**
- 替换原有的 `prevPage()` 和 `nextPage()` 方法
