## 測試清單

### 1. 會話詳情彈窗測試

#### 1.1 基礎功能測試
- [ ] 在「全局會話管理系統」頁面，點擊任意會話行的「詳情」按鈕
- [ ] 驗證彈窗正常打開
- [ ] 驗證「基礎信息」標籤頁顯示以下字段：
  - [ ] 會話 ID（monospace 字體）
  - [ ] 用戶名（藍色標籤）
  - [ ] 創建時間
  - [ ] 過期時間（已過期顯示紅色）
  - [ ] 最後活動時間
  - [ ] 最後請求時間
  - [ ] 刷新次數
  - [ ] 在線時長（格式：X 小時 Y 分鐘）

#### 1.2 網絡信息測試
- [ ] 驗證「網絡信息」區塊顯示：
  - [ ] 首次 IP（monospace 字體）
  - [ ] 當前 IP（monospace 字體）
  - [ ] IP 變更次數（>3 顯示橙色警告標籤）
  - [ ] 設備變更次數（>2 顯示橙色警告標籤）

#### 1.3 IP 變化歷史測試
- [ ] 如果會話有 IP 變化歷史，驗證顯示「IP 變化歷史」區塊
- [ ] 驗證使用時間線格式展示
- [ ] 驗證每個 IP 記錄顯示時間戳和 IP 地址

#### 1.4 設備信息測試
- [ ] 驗證「設備信息」區塊顯示：
  - [ ] 首次設備
  - [ ] 當前設備
  - [ ] 設備指紋（monospace 字體）

#### 1.5 安全信息測試
- [ ] 找一個可疑會話，驗證顯示橙色警告框
- [ ] 驗證警告框顯示可疑原因
- [ ] 找一個已撤銷會話，驗證顯示紅色警告框
- [ ] 驗證警告框顯示撤銷原因和撤銷時間

## 測試環境要求

### 瀏覽器支持
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### 後端 API 要求
確保以下 API 端點正常工作：
- `GET /admin/user-sessions/list` - 會話列表
- `GET /admin/user-sessions/{session_id}` - 會話詳情
- `GET /admin/user-sessions/{session_id}/activity` - 活動時間線
- `POST /admin/user-sessions/{session_id}/flag` - 標記可疑
- `POST /admin/user-sessions/{session_id}/revoke` - 撤銷會話
- `GET /admin/user-sessions/stats` - 統計數據（支持 start_date 和 end_date 參數）

## 快速測試腳本

如果需要快速驗證核心功能，可以按以下順序測試：

1. **基礎功能（5分鐘）**

   - 點擊任意會話的「詳情」按鈕

   - 切換到「活動時間線」標籤，驗證時間線顯示正常

   - 標記一個會話為可疑
   - 驗證狀態更新
   - 取消標記

3. **時間篩選（2分鐘）**
   - 點擊「最近7天」快捷按鈕
   - 驗證數據更新
   - 點擊「全部」恢復

總計：約 10 分鐘完成核心功能測試。
